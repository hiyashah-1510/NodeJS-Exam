# 🍽 Recipe Sharing Platform

A Recipe Sharing Platform built using **Node.js, Express.js, MongoDB, EJS, JWT Authentication, Cookies, and Role-Based Access Control**.

## ✨ Features

* User Registration & Login
* JWT Authentication with Cookies
* Admin & User Roles
* Add, Edit & Delete Recipes
* Upload Recipe Images
* View All Recipes
* My Recipes Section
* Recipe Details Page
* Comment System
* Forgot Password
* Responsive Culinary Theme UI

---

## 🛠 Technologies

* Node.js
* Express.js
* MongoDB
* Mongoose
* EJS
* JWT
* BcryptJS
* Multer
* Cookie Parser
